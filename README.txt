##Начало программы
Инициализируем два массива строк и переменную J, со значением 0
### Цикл For 
С помощью цикла For производим перебор элементов первого массива и при помощи логического  оператора if производим проверку длинны элемента массива и если она меньше или равна 3, 
то записываем элемент во второй массив с индексом "J" увеличивая на 1 этот индекс с каждым циклом.
## Выввод массива.
Вывводим массив, печатая его с помощью цикла for.
